## 1. 실행 방법

```bash
# 인트로 화면으로 시작
./bashbeats

# 프로젝트 파일을 직접 열어서 시작
./bashbeats saves/mysong.bbeat
```

> **터미널 크기**: 140×35 이상을 권장합니다.  
> 실행 시 자동으로 터미널 크기를 조정하지만, 지원하지 않는 터미널에서는 수동으로 크기를 맞춰주세요.

## 2. 인트로 화면

실행하면 아래 항목을 선택할 수 있습니다.

| 항목 | 설명 |
|---|---|
| `[+] New project` | 새 프로젝트를 만들어 DAW 에디터로 진입 |
| `[~] Performance mode` | 연주 모드로 진입 (악기 선택 후 키보드 연주) |
| `파일명.bbeat` | 저장된 프로젝트를 불러와 DAW 에디터로 진입 |

**조작:**
- `↑` / `↓` — 항목 이동
- `Enter` — 선택
- 타이핑 — 파일 이름 필터 검색 (부분 문자열, 대소문자 무시)
- `Backspace` — 검색어 한 글자 삭제
- `^Q` — 종료

## 3. 공통 단축키

어느 모드에서나 동작합니다.

| 키 | 동작 |
|---|---|
| `^F` (Ctrl+F) | FILE 모드로 이동 |
| `^Q` (Ctrl+Q) | 프로그램 종료 |
| `Ctrl+C` | 종료 확인 창 표시 (y/n) |

## 4. TRACK 모드

**기본 화면.** 프로젝트의 트랙 목록을 보여줍니다.

```
 #   Name             Mute    Volume    %    Notes  Base    Instrument
────────────────────────────────────────────────────────────────────────
>1   Lead             [MUTE]  ########  100%  12     C4      piano.wav
 2   Bass                     ####----   50%   8     C3      bass.wav
```

### 단축키

| 키 | 동작 |
|---|---|
| `↑` / `↓` | 트랙 선택 이동 |
| `←` / `→` | 재생 위치(Playhead) 이동 (1박자 단위) |
| `Enter` | 선택한 트랙의 피아노롤 편집 (EDIT 모드) |
| `Space` | 재생 / 일시정지 |
| `m` | 선택 트랙 음소거(Mute) 토글 |
| `+` / `=` | 선택 트랙 볼륨 +5% |
| `-` | 선택 트랙 볼륨 -5% |
| `a` | 트랙 추가 (악기 선택 → 기준음 설정) |
| `d` | 선택 트랙 삭제 |
| `i` | 선택 트랙 악기 변경 |
| `b` | 선택 트랙 기준음(Base Note) 변경 |
| `^F` | FILE 모드 |
| `^Q` | 종료 |

### 재생 진행 바

화면 하단에 재생 위치가 표시됩니다.

```
[====|--------------------]  Tick:8    Bar:3    PLAY
```

- `←` / `→` 로 재생 위치를 이동할 수 있습니다 (재생 중에도 가능).

## 5. EDIT 모드 (피아노롤)

TRACK 모드에서 트랙을 선택하고 `Enter`를 누르면 진입합니다.

```
     .  1  .  2  .  3  .  4  .  5
B4   .  .  .  .  .  .  .  .  .  .
A#4  .  .  .  .  .  .  .  .  .  .
A4   .  [#][ ].  .  .  .  .  .  .   ← 노트
G#4  .  .  .  .  .  .  .  .  .  .
G4   .  .  .  [#].  .  .  .  .  .
```

왼쪽 사이드바에 트랙 목록이 표시됩니다.  
하단 Info Bar에 현재 커서 위치 정보(음이름, MIDI 번호, Tick, Bar, BPM)가 표시됩니다.

### 커서 이동

| 키 | 동작 |
|---|---|
| `↑` / `↓` | 음 높낮이 이동 (반음 단위) |
| `←` / `→` | 시간축 이동 (1 tick) |
| `Ctrl+→` | 마지막 노트 위치로 점프 |
| `Ctrl+←` | 첫 번째 노트 위치로 점프 |
| `[` | 이전 트랙으로 전환 |
| `]` | 다음 트랙으로 전환 |

### 노트 입력

| 키 | 동작 |
|---|---|
| `Enter` | 현재 위치에 1/4박자 노트 배치 / 삭제 (토글) |
| `,` | 긴 노트 시작점 마크 (Span 시작) |
| `.` | 긴 노트 끝점 확정 → 해당 길이의 노트 배치 |
| `Delete` | 현재 위치의 노트 삭제 |

**긴 노트 입력 예시:**
1. 원하는 시작 위치로 커서 이동
2. `,` 키 → 시작점 마크됨
3. `→` 키로 원하는 길이만큼 이동
4. `.` 키 → 노트 배치 완료

### 재생

| 키 | 동작 |
|---|---|
| `Space` | 현재 커서 위치부터 재생 / 일시정지 |
| `↑` / `↓` / `←` / `→` | 재생 중 커서 이동 시 재생 위치도 함께 이동 |

> 재생 중에는 피아노롤이 플레이헤드를 따라 자동으로 횡스크롤됩니다.

### BPM 조절

| 키 | 동작 |
|---|---|
| `+` / `=` | BPM +5 |
| `-` | BPM -5 |

### 화면 복귀

| 키 | 동작 |
|---|---|
| `ESC` | TRACK 모드로 돌아가기 |

## 6. FILE 모드

어느 모드에서나 `^F`로 진입합니다.

```
┌──────────────────────── FILE MANAGER ────────────────────────┐
│  Title   MySong                                               │
│  Path    saves/MySong.bbeat                                   │
│  Stats   3 tracks   42 notes   128 BPM                       │
├───────────────────────────────────────────────────────────────┤
│  ACTIONS                                                      │
│   S     Save           Save project to current path          │
│   L     Load           Browse saves/ and open a project      │
│   N     New            Create new empty project              │
│   R     Rename path    Change save/load file path            │
│   T     Set title      Rename this project                   │
│   E     Export WAV     Render project as a WAV file          │
│   ESC   Back           Return to previous mode               │
└───────────────────────────────────────────────────────────────┘
```

| 키 | 동작 |
|---|---|
| `S` 또는 `^S` | 현재 경로에 저장 |
| `L` | `saves/` 폴더에서 파일 선택하여 불러오기 |
| `N` | 새 프로젝트 생성 (현재 작업 내용 소멸 주의) |
| `R` | 저장 경로 변경 (직접 입력) |
| `T` | 프로젝트 제목 변경 |
| `E` | 현재 프로젝트를 WAV로 내보내기 (`.bbeat` 경로의 확장자를 `.wav`로 변경) |
| `ESC` | 이전 모드로 복귀 |

> 저장 파일은 `saves/` 폴더에 `.bbeat` 확장자로 저장됩니다.

## 7. 연주 모드 (Performance Mode)

인트로 화면에서 `[~] Performance mode`를 선택하면 진입합니다.  
진입 시 악기 선택 화면이 먼저 표시됩니다.

### 키보드 레이아웃

실제 피아노처럼 **흰 건반(아래 행)** 바로 위에 **검은 건반(위 행)** 이 배치됩니다.

```
숫자 행:  [2]C# [3]D#  --  [5]F# [6]G# [7]A#   ← 높은 옥타브 검은 건반
QWERTY:   [q]C  [w]D  [e]E [r]F  [t]G  [y]A  [u]B  ← 높은 옥타브 흰 건반

홈 행:   [s]C# [d]D#  --  [g]F# [h]G# [j]A#   ← 낮은 옥타브 검은 건반
하단 행:  [z]C  [x]D  [c]E [v]F  [b]G  [n]A  [m]B  ← 낮은 옥타브 흰 건반
```

### 단축키

| 키 | 동작 |
|---|---|
| `↑` | 옥타브 +1 (최대 옥타브 7) |
| `↓` | 옥타브 -1 (최소 옥타브 0) |
| `i` | 악기 변경 |
| `ESC` | 인트로 화면으로 돌아가기 |
| `^Q` 또는 `^C` + `y` | 프로그램 종료 |

> **오디오 엔진**: 선택한 WAV 악기를 즉시 로드해 키보드 연주에 사용합니다.<br>
> 로컬 재생은 `aplay`가 있을 때 자동으로 연결되고, TCP 스트리밍과 WAV export는 자체 PCM 렌더러를 사용합니다.

## 8. 프로젝트 파일 형식

`.bbeat` 파일은 일반 텍스트 형식입니다. 직접 편집도 가능합니다.

```
# BashBeats project file
TITLE  MySong
BPM    128

TRACK 0
  NAME   Lead
  INSTR  samples/piano.wav
  BASE   60
  VOL    1.00
  MUTE   0
  # start_tick  note  duration_tick  velocity
  0    60  4   80
  4    62  4   80
  8    64  8   90
END

TRACK 1
  NAME   Bass
  INSTR  samples/bass.wav
  BASE   48
  VOL    0.80
  MUTE   0
  0    48  16  70
END
```

| 필드 | 설명 |
|---|---|
| `TITLE` | 프로젝트 이름 |
| `BPM` | 템포 (20 ~ 300) |
| `TRACK n` | n번 트랙 시작 |
| `NAME` | 트랙 이름 |
| `INSTR` | 악기 WAV 파일 경로 |
| `BASE` | 기준 MIDI 노트 번호 (C4=60, 타악기=-1) |
| `VOL` | 볼륨 (0.00 ~ 1.00) |
| `MUTE` | 음소거 (0=재생, 1=음소거) |
| 노트 행 | `시작tick  노트번호  길이tick  벨로시티` |
| `END` | 트랙 끝 |

**Tick 단위**: 4분음표 = 4 tick (TICKS_PER_QN = 4)

## 9. 빌드

```bash
# 기본 빌드 (실제 오디오 엔진 포함)
make

# Stub 빌드 (오디오 엔진 없이, UI 테스트용)
make stub

# 빌드 결과물 정리
make clean

# 오디오 엔진 디버그 로그 확인 (별도 터미널에서)
make log
# 또는
tail -f /tmp/bashbeats.log
```

**필요 라이브러리:**
- `ncurses` — TUI 렌더링
- `pthread` — 멀티스레드 (오디오 렌더, TCP 스트리밍)
- `libm` — 수학 함수 (피치 시프트용)
- `alsa-utils` — `aplay`를 통한 로컬 오디오 출력

```bash
# Ubuntu/Debian
sudo apt install libncurses-dev alsa-utils

# macOS
brew install ncurses
```
